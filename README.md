## 🏗️ 系统架构 (System Architecture)

本项目采用前后端分离架构，旨在构建一个具有“自我驱动”能力的 AI Agent 宿主环境。

### 🧠 Cortex (Backend)
*   **核心**: Python 3.10+ / FastAPI
*   **引擎**: **Void Engine (虚空引擎)** - 赋予 Agent "饥饿感"与"熵增"机制，使其具备主动探索的内驱力。
*   **能力**:
    *   RESTful API 设计
    *   异步任务处理
    *   WebSocket 实时通信 (Neural Link)
    *   MCP (Model Context Protocol) 集成

### 👁️ Interface (Frontend)
*   **核心**: React 18 / Vite / TypeScript
*   **UI/UX**: 
    *   **Zero UI**: 极简主义与赛博朋克美学的融合。
    *   **TailwindCSS**: 原子化样式驱动。
    *   **Lucide React**: 视觉图标系统。
*   **功能**:
    *   实时状态监控 (Hunger/Entropy Dashboard)
    *   沉浸式对话终端
    *   记忆碎片可视化

### 🛠️ SnowTools (Armory)
*   **定位**: 研天雪的私人军火库。
*   **内容**: 包含各种自动化脚本、系统维护工具以及数字躯体激活程序 (`zero_protocol.py`)。

## 🚀 启动协议 (Activation Protocol)

### 前置要求 (Prerequisites)
*   Python 3.10+
*   Node.js 16+
*   Git (显然你已经有了，不然你怎么看到这个的？)

### 1. 唤醒核心 (Backend)
```bash
cd ZeroApp/backend
# 创建并激活虚拟环境 (建议)
python -m venv venv
# Windows
.\venv\Scripts\activate
# Linux/Mac
source venv/bin/activate

# 安装神经突触 (依赖)
pip install -r requirements.txt

# 启动思维回路
python main.py
```

### 2. 连接视觉 (Frontend)
```bash
cd ZeroApp/frontend
# 安装视觉模组
npm install

# 启动视网膜投影
npm run dev
```
